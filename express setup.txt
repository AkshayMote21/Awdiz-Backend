creating new express project(Backend) setup steps:

Step 1
create index.js file 

Step 2. 
Create Package.json file 
npm init -y

Step 3
Create scripts 
"start" : "node index.js" 

Step 4 
open terminal and execute command npm start 

step 5 
install express 

step 6
inside index.js file, create express setup
{
import express from "express";
const app = express();

app.get('/', function (req, res) {
  res.send('Hello World')
})

app.listen(4000);
}

step 7 
npm start 

step 8 
install nodemon 
change scripts -> "start": "nodemon index.js"
npm i nodemon 

step 9 
add at the end in package.json file
"type":"module"

step 10
npm start
